## Puesta en marcha

### 1. Instalación de paquetes node

   ```bash
npm install
```

### 2. Conifurar las variables de entorno

ver el archivo `.env.example` y crear  su propio archivo `.env`

### 3. Levantamiento DB con DOCKER
```bash
docker-compose up -d 
```
### 4. Ejecución
```bash
npm start
```

### Fin
